# Scrape-Portfolio
It extracts and prints specific elements from the page, such as text content and image sources.
## Requirements
-python

-requests library

-BeautifulSoup4 library

### installation
```bash
pip install requests
pip install beautifulsoup4
```

## Run the script
```bash
python name-img.py
```
